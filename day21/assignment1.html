1. What is HTML?
HTML stands for Hyper Text Markup Language. It is the standard markup language for creating web pages.
2. What is CSS?
CSS stands for Cascading Style Sheets. It is used to style the presentation of HTML elements.
3. What is JavaScript?
JavaScript is a programming language commonly used for creating interactive effects within web browsers.
4. What is the difference between margin and padding?
Margin is the space outside the border of an element, while padding is the space inside the border of an element.
5. What is a div element?
A div element is a block-level container used to group HTML elements for styling purposes.
6. What is the box model in CSS?
The box model is a way of representing the space taken up by an HTML element. It consists of the content area, padding, border, and margin.
7. What is a hyperlink?
A hyperlink, or link, is an element in an HTML document that allows you to navigate to another document or resource.
8. What is the purpose of the alt attribute in HTML?
The alt attribute provides alternative text for an image if the image cannot be displayed.
9. What is a CSS selector?
A CSS selector is a pattern used to select and style one or more elements in an HTML document.
10. What is responsive web design?
Responsive web design is an approach to designing and coding websites to ensure they provide an optimal viewing experience across a wide range of devices and screen sizes.
